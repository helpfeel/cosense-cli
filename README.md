# @helpfeel/cosense-cli

Cosenseのページを読む・調べるCLIとAgent Skill。ページを書く機能はまだ実装できていない

## Install

### CLI

```bash
npm install -g https://github.com/nota/cosense-cli
cosense --help
```

### Claude Code Agent Skill

```
/plugin marketplace add nota/cosense-cli
/plugin install cosense-cli@cosense-cli
```

Skillの実行にはCLIが必要

## 開発

- install [direnv](https://direnv.net/)
- run `npm install`
- run `claude` or `codex`
